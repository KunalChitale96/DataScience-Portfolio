Information about this project

This is a project about predicting height vs weight using linear regression model.
In this,as we can see the figure on the left shows the predicted values and plotted values form a perfect slanting line which is ideal for regression model. Although in real life it is not possible to see that all the points
ie Height and Weight fall on the line itself, it happens most of the time that there are some points above the line and very few fall below the line.
We can use cost function to find errors and determine best possible line which encompasses all the values that are plotted on the graph, but as this is my first project it is an ametuer attempt to find out more about linear regression model and how it can be used to predict values in the future.

Key points
  • As height increases so does the weight.
  • Predicted values are shown along the blue line.
